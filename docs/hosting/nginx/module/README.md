# NGiNX Module

[Zu NGiNX zurück](https://wiki.page-speed.ninja/hosting/nginx/)

Im Folgenden finden Sie eine Liste mit Modulen von Drittanbietern für NGINX und NGINX Plus, die von Mitgliedern der NGINX-Community erstellt und verwaltet werden. NGINX, Inc. bietet keine Unterstützung für diese Module. Wenden Sie sich daher bitte an jeden einzelnen Modulentwickler, wenn Sie Fragen oder Hilfe haben. Informationen zum Hinzufügen eines Moduls zu dieser Liste finden Sie unter [https://github.com/nginxinc/nginx-wiki](https://github.com/nginxinc/nginx-wiki). Eine Liste der offiziell unterstützten Module von NGINX, Inc. und von NGINX Plus zertifizierten Modulen unserer Partner finden Sie unter [https://www.nginx.com/products/nginx/modules](https://www.nginx.com/products/nginx/modules).
